# DataPlatform - Next.js Application

This is a modern, high-performance web application built with Next.js 14, TypeScript, and Tailwind CSS. It features a completely bespoke layout spanning a heroic background canvas animation, responsive grids, and highly-optimized CSS-only interactive elements completely bypassing React renders where possible.

## Getting Started

First, install all the project dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment on Vercel

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com/new).

Since this repository contains a `vercel.json` declaring the framework, Vercel will automatically detect the settings:
1. Push this repository to GitHub/GitLab/Bitbucket.
2. Import the project into Vercel.
3. Vercel will automatically detect `nextjs` framework and install dependencies.
4. Click Deploy!
